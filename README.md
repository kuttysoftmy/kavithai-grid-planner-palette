# ⚡ Kavithai Grid Planner & Palette Studio

> Built autonomously by **Angela AutoPilot** based on [quotes-image-generator](https://github.com/kuttysoftmy/quotes-image-generator).

A visual workspace and content planner designed to complement the Tamil Quotes Image Generator. Instead of creating isolated single images, this tool allows content creators to plan multi-post grids (like Instagram 3x3 structures) for their Kavithais (poems), curate custom color palettes, organize collections of quotes in an offline workspace (IndexedDB), and batch-push draft texts directly to the Quotes Generator canvas to streamline their social media workflow.

## Tech Stack
React + Tailwind CSS + Vite + DnD Kit + IndexedDB

## Quick Start
```bash
npm install
npm build
```
